泛型对象

可以用 interface 或 type 加上类型参数来声明泛型对象，其中前者只能是 object，后者则可以是所有类型。

泛型对象常常用于容器，如 Array，Map，Set 等。

泛型容器可以是只读的，表明该容器的元素不能被修改。
