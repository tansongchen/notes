厨师培训教材

厨师培训教材​

概述​

烹调工序​

  1. 原料选择
  2. 初步加工：宰杀、择剔、剖剥、拆卸、泡发、洗涤
  3. 刀工刀法：改切成各种形状，如条、段、片、丁、块等
  4. 配菜：将各种原料按一定比例搭配
  5. 火候：火力的大小、加热时间的长短
  6. 调味

选料​

鸡蛋新鲜度的鉴别​

  1. 看、摸、摇：蛋壳不光亮、手感粗糙、轻摇蛋内不晃动
  2. 灯光照：对灯光照看，蛋内呈淡而均匀的红色
  3. 盐水泡：按 10:1 配制盐水，能沉底
  4. 磕开：蛋黄完整、蛋清黏稠

畜肉新鲜度的鉴别​

  1. 看外表：有一层微微干燥的膜，色泽光润，切片呈淡红色，肉质透明
  2. 触硬度：紧实有弹性，按后很快复原
  3. 闻气味：无异味
  4. 察脂肪：均匀分布，保持固有色泽，硬实

冷冻畜肉新鲜度的鉴别​

  • 肉表滋润，保持原色
  • 肌肉断面呈淡玫瑰色
  • 组织坚实
  • 关节色白
  • 无异味
  • 无杂色

光禽新鲜度的鉴别​

  1. 察嘴喙：干燥、有光泽及弹性，无异味
  2. 看眼睛：眼球充满眼窝，角膜有光泽
  3. 审皮肤：呈淡黄色或白色，表面干燥，无异味
  4. 触肌肉：肌肉结实而有弹性，料面为玫瑰色
  5. 看脂肪：脂肪色白或带淡黄

鱼类新鲜度的鉴别​

  1. 察鱼鳃：鳃盖紧闭，黏液少并呈透明状；鳃质鲜红或粉红，无异味
  2. 看鱼眼：眼求完整，稍凸出，澄清透明（不充血、不发红）
  3. 审鱼体：体面清洁，鱼鳞完好紧密有光泽，肌体有弹性，肛门周　呈圆坑形，肚腹不鼓胀

原料初步加工​

初步加工的标准​

  1. 保持营养成分
  2. 保证清洁卫生
  3. 使原料各部分得到合理使用
  4. 保持原料的完整性
  5. 保证菜肴的质、色、味、形不受影响

刀工基本知识​

刀工的基本要求​

  1. 与烹饪结合：
    1. 汆爆火大时间短，原料薄小；炖焖火小时间长，原料厚大
    2. 质地软的比质地硬的要大一些
  2. 整齐、均匀、利落
  3. 主料辅料形状调和

运刀方法​

  1. 身体姿势：两腿分开，双肩放平，上身前倾，头部硝低
  2. 右手拇指与食指握住刀根部，手掌握住刀柄
  3. 左手掌心向下，五指自然弯曲按住原料，中指第一关节抵住刀身，然后有节奏地向后等距离匀速移动

刀法分类​

  1. 切：直切、推切、拉切、锯切、铡切、滚刀切
  2. 片：坡刀片、推刀片、拉刀片
  3. 剁

配菜基本知识​

配菜的数量要求​

  • 有主辅料之分：主料要比辅料多才能突出主料
  • 无主辅料之分：各种应该一样多

配菜的口味要求​

  • 以主料的口味为主，辅料的口味适应主料的口味：例如鲜味肉类 + 鲜味菜
  • 主料口味较淡，以鲜味较重的辅料及调料弥补：例如海参 + 鲜汤
  • 主流口味过浓或过腻，加适当辅料调和或冲淡

配菜的质地要求​

  • 辅料的质地与主料相同：脆配脆，软配软
  • 辅料的质地不同于主料

配菜的形状要求​

  • 块配块，片配片，丝配丝，丁配丁
  • 辅料略小于主料

配菜的颜色要求​

  • 顺色：主辅料采用一种颜色
  • 花色：主辅料颜色不同

火候基本知识​

烧​

指将主料改刀后，过油或水烫；再用辅料炝锅，放入调料、汤和主料，烧到一定程度出锅：

  • 红烧：加糖色或老抽使色泽红润
  • 干烧：加汤汁并用文火收干

烤​

以辅射热为主的烹制方式，可以分为暗炉烤、明炉烤、烤箱烤等。

焯水​

作用​

  • 肉：去除血污和腥膻气味
  • 蔬菜：色泽鲜艳、口味脆嫩，并去除异味
  • 缩短正式烹调时的加热时间
  • 调整不同性质的原料成熟时间

分类​

  • 冷水锅：即把原料与冷水同时下锅，用于腥味重、血污多的原料
    • 操作要点：经常翻动，使受热均匀
  • 沸水锅：先将水煮沸再下锅，用于腥味小、血污少的原料
    • 操作要点：水略沸即可取出，加热时间不能过长

原则​

  • 根据原料不同的性质适当掌握焯水时间
  • 特殊气味的原料应该分开焯水

走红​

即使原料上色，主要适用于烧、焖、煨等菜

  • 卤汁走红：将原料放入锅中，加入酱油、绍酒、糖和水等，先用旺火烧沸，再用小火加热使调味品的色泽进入原料
    • 要点：掌握好颜色的浓度
  • 过油走红：将有色调味品抹在原料表面，再下入油锅中炸至原料上色
    • 要点：表面调味品要均匀
