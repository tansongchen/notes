C++

📄️ async 与 future

async 接收一个函数和其参数，返回一个 future 类模板对象，代表这个函数在将来会返回值；当调用 .get() 方法时主线程阻塞直到异步线程结束。

📄️ 互斥锁

针对一个可能被多个[[线程]]使用的资源，定义一个变量 m，进入[[临界区]]之前必须对 m 上锁，退出时解锁，而它上锁之后就不能被别的线程再次上锁，从而保证不会出现访问冲突。

📄️ 原子类型

在 C++ 中，可以把共享数据定义成 std::atomic，每次访问时自动加锁和解锁

📄️ 线程

操作系统能够进行 CPU 调度的最小单位，包含在[[进程]]之中。

📄️ 线程阻塞

等待另一个线程执行完之后，再执行本线程的其余部分。
