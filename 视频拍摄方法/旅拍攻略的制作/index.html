旅拍攻略的制作

1. 基本结构​

分为多个场景，每个场景通过几个镜头来讲述一个小故事

如果没有时间构思脚本，只需要想一想会去哪里旅游即可

2. 调性​

  • 慢节奏视频（画面停留长，运镜幅度小）：使用 60 帧或以上进行拍摄，方便升格处理
  • 快节奏视频（画面停留短，运镜幅度大）：尽量选择手持拍摄

3. 构建多景别​

每个场景使用多种景别，增加画面的可看性

  • 近景：人本身
  • 中景：人的动作
  • 全景：交待场景

4. 空镜​

不能只拍景色，也不能只拍人物

  • 大景别空景：日落日出等，向观众介绍场景
  • 小景别空景：把美好的事物记录下来，作为转场画面

5. 构图和运镜​

简化构图，突出主体，内容能少则少

6. 加入个人色彩​

  1. 根据旅行攻略大致写好旅拍模板
  2. 在小红书等网站上找一些图片，提前找到拍摄机位和灵感
  3. 尽量在日出后一小时、日落前一小时进行拍摄
  4. 构思旅行的故事
    1. 旅行的收获
    2. 旅途中的启发
    3. 旅行的原因
    4. ……
  5. 构思开场和结尾 Vlog 开场和结尾思路
  6. 构思转场 Vlog 转场思路
